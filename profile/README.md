# MadByAd

Hello this is MadByAd 👋 i am an Indonesian developer who has passion for code and arts but mostly make games!

- 🤵 This is MadByAd and welcome to my developer Github Profile! where i ocassionaly publish repositories related to my game development
- 👩‍💻 This github organization is used to host my developer website [madbyad.my.id](https://madbyad.my.id) which you can learn more about me there!
- 🎮 Most of my games are available on my [itch.io](https://madbyad.itch.io) page but i also have a [playstore](https://playstore.madbyad.my.id) page!
- 🍿 Here some cool fact about me -  the name MadByAd is simply stands for Made By Aditya 😅 i choose the name since its quite simple and has a techy sounding and an eastern europe-like vibes for me
- 🧙 Also, my real name is Aditya Wahyu Santoso, i am the man behind this developer identity! you can learn more about it trough my personal website [adit.madbyad.my.id](https://adit.madbyad.my.id) consider taking a visit!
